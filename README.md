Code to enable reproduction of figures from "Spectrally Robust Covariance Shrinakge of Hotelling's T^2 in High Dimensions" by B. Robinsoin and V. Latimer, as well as additional testing of the algorithm if desired.

###Instructions to run###

* For the ROC curves Figures 4 and 5 in the main document, run the script EnergyDetectors.py (once uploaded).  
* For the ROC curve Figure 6, run the script crawdad.py (once uploaded).
* For the ROC curves in the separate Supplemental Material, run EnergyDetectors.py.  For the heavy-tail simulations, choose options 2 or 3 at the beginning, then follow the prompts.

All ROC curves were generated by running 3000 independent trials, with dimension p = 200, and number of training samples per trial being either 300 or 100.

Once the main scripts are uploaded, there will likely be additional options to simply press one button for the figures appearing in the paper and supplemental material.
